# WinForms Application

A Windows Forms application generated by AGP CLI.

## Requirements

- Windows OS
- .NET 9.0 or later

## Build and Run

```bash
dotnet build
dotnet run
```

## Publish

```bash
dotnet publish -c Release -o ./publish --self-contained true -r win-x64
```

## Features

- Modern WinForms UI
- .NET 9.0 targeting
- Ready for customization
