# Web API Application

A REST API application generated by AGP CLI.

## Build and Run

```bash
dotnet build
dotnet run
```

The API will be available at:
- HTTP: http://localhost:5000
- HTTPS: https://localhost:5001

## Swagger Documentation

Navigate to: https://localhost:5001/swagger

## API Endpoints

### GET /api/values
Get all values

### GET /api/values/{id}
Get a specific value by ID

### POST /api/values
Create a new value

### PUT /api/values/{id}
Update a value

### DELETE /api/values/{id}
Delete a value

## Example Usage

```bash
# Get all values
curl http://localhost:5000/api/values

# Get specific value
curl http://localhost:5000/api/values/1

# Create a value
curl -X POST http://localhost:5000/api/values \
  -H "Content-Type: application/json" \
  -d '"new value"'
```

## Publish

```bash
dotnet publish -c Release -o ./publish
```
